# p 134

```clojure
(fn [k hv] (if (contains? hv k)
             (if (nil? (get hv k))
               true
               false)
             false))
```

이럴 땐 `and`를 쓰면 좋다.

```clojure
(fn [k hv]
  (and (contains? hv k) (nil? (get hv k))))
```
